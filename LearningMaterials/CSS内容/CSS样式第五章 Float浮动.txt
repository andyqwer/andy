css float（浮动）的基础知识
css float（浮动）概念（以下4条）：
1.浮动会使元素向左或向右移动，只能左右，不能上下。
2.浮动元素碰到包含框或另一个浮动框，浮动停止。
3.浮动元素之后的元素将围绕它，之前的不受影响。
4.浮动元素会脱离标准流。
注：第3条补充：浮动元素脱离标准流（文档流）后，其后面的元素会占据其空间；而因为浮动元素未脱离文本流，所以其后面元素中的文本因不能被浮动元素遮挡，将围绕浮动元素

float有四个属性，分别是：
float：none；  没有浮动；
float：left；  左浮动；
float：right； 右浮动；
float：inherit；继承父元素的浮动；

浮动（float）的概念：
2.浮动元素碰到包含框（父元素的左边框或者右边框）或另一个浮动框，浮动停止。
解释：在三个相邻块元素下，第一个浮动元素在本行找到父元素的左边框或右边框停止，第二个浮动元素在本行（第二个元素会上移，占据第一个浮动元素的位置，与第一个浮动元素同行）找到第一个浮动元素紧挨其后停止。第三个浮动元素在本行（第三个元素会上移，占据第一个浮动元素的位置，与第一个浮动元素同行）找到第二个浮动元素紧挨其后停止。

清除浮动
1. 在浮动元素后面，添加div，样式设置为clear：both
2. 给浮动元素的容器添加overflow：hidden;zoom:1
3.给浮动元素的容器添加
clearfix：after伪类 样式为
content："."
;height:0;
display:block;
visibly:hidden;
clear:Both;
4： 父级元素定义height。只适合高度固定的布局。
5： 父级元素也一起浮动。不推荐，会产生新的浮动问题。
